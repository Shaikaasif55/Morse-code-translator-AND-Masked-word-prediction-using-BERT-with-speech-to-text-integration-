## Overview

This repository contains two Python-based Natural Language Processing (NLP) applications demonstrating text processing and transformer models.

## Projects

### 1. Morse Code Translator

A Python program that converts English text into Morse code and decodes Morse code back into readable text.

#### Features

- Text to Morse conversion
- Morse to Text conversion
- Simple command-line interface
- Fast encoding and decoding

---

### 2. Masked Word Prediction

A transformer-based NLP application that predicts missing words in sentences using the DistilBERT pretrained language model.

#### Features

- Context-aware word prediction
- Uses Hugging Face Transformers
- Based on DistilBERT
- Easy-to-use inference

---
